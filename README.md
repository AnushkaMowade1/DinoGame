# 🦖 Dino Game – High Performance Browser Game

A fast, responsive browser-based game featuring smooth animations, real-time obstacle generation, and adaptive difficulty. Built with a focus on performance, clean architecture, and engaging gameplay.

---

## 🚀 Features

- ⚡ 60 FPS smooth gameplay with optimized game loop and rendering  
- 🎯 Accurate collision detection for precise gameplay  
- 📈 Adaptive difficulty that increases challenge over time  
- ⌨️ Responsive keyboard controls  
- 📱 Fully responsive across devices  
- ⚡ Fast load time (<1s)

---

## 🛠️ Tech Stack

- JavaScript  
- HTML5  
- CSS3  
- Vercel (Deployment)
